# Welcome to your Expo app 👋

## English

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

### Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

### Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```

This command will move the starter code to the **app-example** directory and create a blank **app** directory where you can start developing.

To learn more about developing your project with Expo, look at the following resources:

- [Expo documentation](https://docs.expo.dev/): Learn fundamentals, or go into advanced topics with our [guides](https://docs.expo.dev/guides).

Join our community of developers creating universal apps.

- [Expo on GitHub](https://github.com/expo/expo): View our open source platform and contribute.

## Русский

Добро пожаловать в ваше приложение Expo 👋

### Начало работы

1. Установите зависимости

   ```bash
   npm install
   ```

2. Запустите приложение

   ```bash
   npx expo start
   ```

В выводе вы найдете варианты открытия приложения в

- [сборке для разработки](https://docs.expo.dev/develop/development-builds/introduction/)

### Получить свежий проект

Когда вы будете готовы, запустите:

```bash

```

Эта команда переместит стартовый код в каталог **app-example** и создаст пустой каталог **app**, где вы сможете начать разработку.

## Қазақша

Сіздің Expo қолданбаңызға қош келдіңіз 👋

Бұл [`create-expo-app`](https://www.npmjs.com/package/create-expo-app) көмегімен жасалған [Expo](https://expo.dev) жобасы.

### Бастау

1. Тәуелділіктерді орнатыңыз

   ```bash
   npm install
   ```

2. Қолданбаны іске қосыңыз

   ```bash
   npx expo start
   ```

Шығарылымда қолданбаны ашу опцияларын таба аласыз

- [дамыту құрастыруында](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android эмуляторында](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS симуляторында](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), Expo көмегімен қолданбаны әзірлеуді сынауға арналған шектеулі құмсауыт

**app** каталогындағы файлдарды өңдеу арқылы әзірлеуді бастай аласыз. Бұл жоба [файлға негізделген маршруттауды](https://docs.expo.dev/router/introduction) пайдаланады.

### Жаңа жобаны алыңыз

Дайын болған кезде, іске қосыңыз:

```bash
npm run reset-project
```

Бұл команда бастапқы кодты **app-example** каталогына жылжытады және әзірлеуді бастауға болатын бос **app** каталогын жасайды.

### Толығырақ

Expo көмегімен жобаңызды әзірлеу туралы көбірек білу үшін келесі ресурстарды қараңыз:

- [Expo құжаттамасы](https://docs.expo.dev/): Негіздерді үйреніңіз немесе біздің [нұсқаулықтармен](https://docs.expo.dev/guides) кеңейтілген тақырыптарға кірісіңіз.
- [Expo оқулығы](https://docs.expo.dev/tutorial/introduction/): Android, iOS және веб-сайтта жұмыс істейтін жобаны жасайтын қадамдық оқулықты орындаңыз.

### Қауымдастыққа қосылыңыз

Әмбебап қолданбалар жасайтын әзірлеушілер қауымдастығына қосылыңыз.

- [GitHub-тағы Expo](https://github.com/expo/expo): Біздің ашық бастапқы платформамызды қараңыз және үлес қосыңыз.
- [Discord қауымдастығы](https://chat.expo.dev): Expo пайдаланушыларымен сөйлесіңіз және сұрақтар қойыңыз.
